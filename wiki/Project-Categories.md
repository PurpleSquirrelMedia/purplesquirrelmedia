# Project Categories

This page organizes all Purple Squirrel Media repositories by category for easy navigation.

## NFT Infrastructure

Projects related to NFT creation, minting, and distribution on Solana.

| Repository | Description | Tech Stack |
|------------|-------------|------------|
| [`candymachine`](https://github.com/PurpleSquirrelMedia/candymachine) | Production-ready Candy Machine V2 responsive UI with multi-mint support | React, TypeScript |
| [`solana-candymachine-v2-gumdrop`](https://github.com/PurpleSquirrelMedia/solana-candymachine-v2-gumdrop) | Candy Machine with Gumdrop airdrop functionality | TypeScript |
| [`Metaplex`](https://github.com/PurpleSquirrelMedia/Metaplex) | Full Metaplex protocol implementation | Rust, TypeScript |
| [`NFT`](https://github.com/PurpleSquirrelMedia/NFT) | Core NFT tooling and utilities | Rust, JavaScript |
| [`solana-nft-gallery`](https://github.com/PurpleSquirrelMedia/solana-nft-gallery) | NFT gallery showcase application | TypeScript |

### Features Supported
- Public mint with countdown
- Civic gatekeeper integration
- Whitelist & presale support
- End date/number settings
- SPL token minting
- Multi-mint in single transaction
